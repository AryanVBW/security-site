XMLBeans XML Entity Expansion

CVE-2021-23926 [CVE json] [OSV json]

Last updated: 2021-01-14T14:45:47.410Z

Affected

  • Apache XMLBeans from Apache XMLBeans through 2.6.0

Description

The XML parsers used by XMLBeans up to version 2.6.0 did not set the properties needed to protect the user from malicious XML input. Vulnerabilities include possibilities for XML Entity Expansion attacks. Affects XMLBeans up to and including v2.6.0.

References

  • https://poi.apache.org/
  • https://issues.apache.org/jira/browse/XMLBEANS-517

A carefully crafted TNEF file can cause an out of memory exception

CVE-2022-26336 [CVE json] [OSV json]

Last updated: 2022-03-04T15:58:59.057Z

Affected

  • poi-scratchpad from unspecified through 5.2.0

Description

A shortcoming in the HMEF package of poi-scratchpad (Apache POI) allows an attacker to cause an Out of Memory exception. This package is used to read TNEF files (Microsoft Outlook and Microsoft Exchange Server). If an application uses poi-scratchpad to parse TNEF files and the application allows untrusted users to supply them, then a carefully crafted file can cause an Out of Memory exception. This issue affects poi-scratchpad version 5.2.0 and prior versions. Users are recommended to upgrade to poi-scratchpad 5.2.1.

References
