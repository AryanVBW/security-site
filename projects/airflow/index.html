Hive Provider RCE vulnerability with hive_cli_params

CVE-2022-46421

Affected

  • Apache Airflow Hive Provider versions before 5.0.0

Description

Improper Neutralization of Special Elements used in a Command (‘Command Injection’) vulnerability in Apache Software Foundation Apache Airflow Hive Provider.

This issue affects Apache Airflow Hive Provider: before 5.0.0.

Description

Improper Neutralization of Special Elements used in a Command (‘Command Injection’) vulnerability in Apache Software Foundation Apache Airflow, Apache Software Foundation Apache Airflow MySQL Provider.

This issue affects Apache Airflow: before 2.5.1; Apache Airflow MySQL Provider: before 4.0.0.
